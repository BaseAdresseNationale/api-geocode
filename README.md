# @etalab/api-geocode

API haute performance de géocodage CSV, propulsée par [addok](https://github.com/addok/addok).

## Utilisation

```bash
# Copier les variables d'environnement
cp .env.sample .env

# Installer les dépendances
yarn

# Lancer le serveur (port 5000 par défaut)
yarn start
```
